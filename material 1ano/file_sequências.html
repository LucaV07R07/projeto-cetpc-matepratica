Progressões Aritméticas (PA)

Uma Progressão Aritmética (PA) é uma sequência numérica em que cada termo, a partir do segundo, é obtido pela adição de uma constante chamada de razão ou diferença com o termo anterior. A razão é representada por 'd', e é a diferença entre cada termo subsequente e o anterior.

Por exemplo, considere a sequência 2, 5, 8, 11, 14. É uma PA com uma razão (d) de 3, pois, para obter cada termo, somamos 3 ao termo anterior. Assim, 5 = 2 + 3, 8 = 5 + 3 e assim por diante.

As progressões aritméticas são amplamente aplicadas em várias áreas, como finanças, física e estatística. Em finanças, são usadas para calcular juros, depreciação e outros cálculos relacionados a investimentos.

A fórmula geral para encontrar o termo 'n' de uma PA é dada por: an = a + (n - 1)d, onde 'a' é o primeiro termo, 'n' é o número do termo e 'd' é a razão.

Além disso, a soma dos primeiros 'n' termos de uma PA (Sn) pode ser calculada usando a fórmula: Sn = (n/2) * (2a + (n - 1)d), onde 'a' é o primeiro termo, 'd' é a razão e 'n' é o número de termos a serem somados.

Progressões Geométricas (PG)

Uma Progressão Geométrica (PG) é uma sequência numérica em que cada termo, a partir do segundo, é obtido pela multiplicação de uma constante chamada razão (r) com o termo anterior.

Por exemplo, considere a sequência 2, 6, 18, 54. É uma PG com uma razão (r) de 3, pois cada termo é obtido multiplicando o termo anterior por 3. Assim, 6 = 2 * 3, 18 = 6 * 3 e assim por diante.

As progressões geométricas são fundamentais em diversas áreas, como crescimento populacional, física, biologia e economia. Por exemplo, o crescimento exponencial pode ser modelado usando uma PG.

A fórmula geral para encontrar o termo 'n' de uma PG é dada por: an = a * r(n - 1), onde 'a' é o primeiro termo, 'n' é o número do termo e 'r' é a razão.

Além disso, a soma dos primeiros 'n' termos de uma PG (Sn) finita pode ser calculada usando a fórmula: Sn = a * (1 - rn) / (1 - r), onde 'a' é o primeiro termo, 'r' é a razão e 'n' é o número de termos a serem somados.
